リンク色を複数設定してみよう

■リンク１「Yahoo!」へリンクする（ページプロパティでの設定）

■リンク２「Google」へリンクする（クラス名：jump1）

■リンク３「MSN」へリンクする（クラス名：jump2）

 

同じHTML上で複数のリンク色を表現するには
リンクタグ「a」に対してクラスの定義を設定する必要があります。

○基本の色変更の設定は

a:link {color:#000000;}
a:visited {color:#000000;}
a:hover {color:#000000;}
a:active {color:#000000;}

○クラスを追加して部分的に定義する方法１

.jump a:link {color:#000000;}
.jump a:visited {color:#000000;}
.jump a:hover {color:#000000;}
.jump a:active {color:#000000;}

○クラスを追加して部分的に定義する方法２

a.jump:link {color:#000000;}
a.jump:visited {color:#000000;}
a.jump:hover {color:#000000;}
a.jump:active {color:#000000;}

場面に応じて必要な定義をします。
