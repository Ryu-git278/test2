divタグでくくってCSSを定義してみよう

divタグでくくって、そのdivタグに定義を適用するには
はじめにdivタグでくくってから定義を作成する方法と
定義を作ってからdivタグでくくって適用する方法があります。

選択した範囲をdivタグでくくるには
くくりたい部分をドラックして選択し、
「挿入」メニュー　→　「レイアウトオブジェクト」　→　「divタグ」　をクリック。

もしくは「挿入」パネルの「Divタグを挿入」ボタンを押す。

「Divタグを挿入」ウィンドウが表示されたら
適用するIDまたはクラス名が分かっている場合は
ID名またはクラス名を所定の場所に記述する。
（その際「.」や「#」は記述しない）
適用するIDまたはクラス名が不明な場合は
何も記述せずに「OK」を押す。

後からタグにCSSを適用する場合は先にCSS定義を作成し、
ステイタスバーの適用したいタグをクリック
その場で右クリックして
「クラス設定」あるいは「ID設定」の中から
適用したいクラス名またはID名を選択する。
