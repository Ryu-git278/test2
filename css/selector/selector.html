●属性セレクタ

○属性が指定されている

#sample5 img[alt] {
  border: 2px solid #ff0000;
}

○属性が指定されていて、値が一致している

#sample6 img[alt="画像2"] {
  border: 2px solid #ff0000;
}

○属性が指定されていて、値に指定された語句（文字ではない）を含んでいる

#sample7 img[alt~="画像"] {
  border: 2px solid #ff0000;
}
